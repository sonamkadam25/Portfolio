Projects

Projects

Below are some of the projects I’ve built using a variety of technologies, including HTML, CSS, JavaScript, React, and C++.

FoodFusion - Food Delivery Application

FoodFusion is a food delivery app built with the MERN stack. It offers features like browsing food by category, adding items to the cart, placing orders, and user authentication with JWT and bcrypt. I focused on the frontend, ensuring a seamless user experience.

Technology - MongoDB, Express, React, Node.js

QuickCart - An E-Commerce Application

QuickCart is an e-commerce platform built with React and Firebase. It features user authentication, product browsing by category, an add-to-cart functionality, and an admin dashboard for managing products, users, and orders. I implemented responsive UI and integrated Redux for efficient state management.

Technology - React, Firebase, Tailwind CSS

SkyInsight - Weather Application

SkyInsight is a web application that provides real-time weather updates based on your location or city search. It displays current temperature, wind speed, humidity, and cloud cover using data from a weather API. Built with HTML5, CSS3, and JavaScript, it features a user-friendly interface and utilizes the Geolocation API for accurate, location-based weather information.

Technology - HTML, CSS, JavaScript, Geolocation API, Weather API


SortVision - Sorting Visualizer

SortVision is a web app that visualizes sorting algorithms like Bubble Sort, Merge Sort, and more. It features color-coded sorting steps, adjustable speed, data size, and random data generation, along with displaying time and space complexity for each algorithm.

Technology - HTML, CSS, JavaScript, C++

Amazon Clone

I built an Amazon Clone using HTML and CSS, replicating the design and layout of Amazon’s homepage. The project includes a responsive navigation bar, product listings, and a clean, user-friendly interface that mimics the real website’s structure.

Technology - HTML, CSS
